TW-Unlocked

This is a userscript that should be loaded via GreaseMonkey.
 You can get the script at: Just click me for the script.

Usage

When you add it go to turbowarp and you should see a button that says "TW-Unlocked".


 Clicking it will open a menu that allows you to load extensions unsandboxed and disable security manager.
 And if you are good with javascript you can look at the TWunlocked object in the console.
